Publishing models in your own HTML page

The Systo approach allows for 4 areas of creativeness:

  1. You can create a model;
  2. You can create a web page to make a model (or many models) available to others;
  3. You can create a widget capable of displaying some aspect of a model or the results of running a model;
  4. You can create a new diagrammatic language.

This web page deals with the second of these 4 topics: making a web page for a model.

You may already be familiar with the idea of seeing models on the web, and running models in your browser for example, isee's NetSim, Forio, and InsightMaker.
