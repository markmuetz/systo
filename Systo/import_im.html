Import an Insight Maker model

This is highly experimental, and currently works only for simple Insight Maker models (no arrays, functions, sketch graphs, tabulated lookups, etc!).

First, paste the Insight Maker XML file into this online XML-to-JSON converter, and convert to JSON.

Then, paste the JSON text into the textarea below, and click the Import button.

The text from a sample model (a koala disease model, by Heather Freeman) is provided as an example. So, just click the Import button, and the model will be converted and displayed as a Systo System Dynamics diagram.

To run the model, simply click the Run button in the Run Control dialog window. You can them move sliders for the stocks and parameters, and observe the effect on model behaviour.
