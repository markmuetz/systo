Import a Vensim model

This is highly experimental, and currently works only for simple Vensim models (no arrays, functions, sketch graphs, tabulated lookups, etc!).

Just paste the contents of a text .mdl file into the textarea below, then click the Import button.

The text from a sample model (the Shodor predator-prey model) is provided as an example. So, just click the Import button, and the model will be converted and displayed as a Systo System Dynamics diagram.

To run the model, simply click the Run button in the Run Control dialog window. You can them move sliders for the stocks and parameters, and observe the effect on model behaviour.
