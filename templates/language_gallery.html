Systo language gallery

This page illustrates some Systo diagramming languages.

Each language is illustrated with a sample model.

system_dynamics

Systo's diagrammatic language for System Dynamics (stock-and-flow) modelling.

system_dynamics_im

Approximation to the System Dynamics notation used by Insight Maker.

biological_pathway

Approximation to the SBGN diagramming notation for representing biological pathways.

wcs_conceptual_modelling

The conceptual modelling language for conservation projects used in the Miradi project management software.
