Model hub

This page gives you access to a variety of Systo-based applications which can let you view, edit and run Systo models.

One way of accessing this page is by searching for Systo models on myExperiment, selecting a particular model, then clicking on the link in the Run field for that model.

Alternatively, you can use this page for any Systo model stored on the web, by entering its URL after the URL for this page, i.e.
http://flask.systo.org/modelhub.html?modelurl=URL for the model you want to load]

Systo page Description
SystoLite SystoLite is a very simple environment for System Dynamics modelling. It allows viewing and editing othe model diagram, and simple displays (graph, table) for viewing simulation results.
